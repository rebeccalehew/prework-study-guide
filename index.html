HTML

  • HTML adds structure to the content of a website.
  • The head element contains information about the webpage.
  • Head elements are not visible to the user.
  • The body element represents the visible content shown to the user.

CSS

  • Defines how webpage content is presented to users - colors, aesthetics, visual layout, organization
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS can be added internally or via an external style sheet linked to the HTML head.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git branch: lists your branches; a * will appear next to the currently active branch
  • git fetch: fetch down all the branches from that Git remote
  • git merge: merge a remote branch into your current branch to bring it up to date
  • git push: Transmit local branch commits to the remote repo branch
  • git pull: fetch and merge any commits from the tracking remote branch

JavaScript

  • Allows users to interact with webpages.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript is read top to bottom -- order matters!
